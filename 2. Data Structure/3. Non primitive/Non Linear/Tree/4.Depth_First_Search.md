# Depth First Search (DFS)- Stack

- **한 드라마를 처음부터 끝까지 한번에 몰아서 봄.**
- **재귀함수 (한놈만 끝까지 패)**
- **경로탐색/ 네트워크/ 조합유형 문제들때 사용**

: 맹목적으로 각 노드를 탐색할 때 주로 사용. 그리고 컴퓨터 구조적으로 항상 스택의 원리를 사용하고 있다.
